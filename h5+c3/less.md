变量：@color: pink;

- 必须以@开头
- 不能以数字开头
- 大小写敏感
- 不能包含特殊字符

less编译：easy-less插件

less嵌套：加&表示同一级，不加则表示后代；

``` 
.header {
    background-color: blue;
    a {
        // 伪类,伪元素
        &:hover {};
        ::before {};
    }
}
```

less运算

- 支持加减乘除
- 运算符两侧必须有空格隔开
- 两个数参与运算，若只有一个数有单位，则结果它为准
- 两个数都有单位，且不一样，则以前一个数为准
