自定义事件

- 一句话，在子组件上通过$emit()触发父组件给子组件绑定的自定义事件
- 自定义事件是给子组件绑定的
- 给子组件绑定一个自定义事件，需要在子组件内部触发
- 可以用修饰符把自定义事件转换为原生事件，比如点击事件：@click.native
- 其实就是给子组件根节点绑定了点击事件，即事件委派

``` 
// 父组件
<template>
  <div id="app">
    <Son @handler="getsum"></Son>
  </div>
</template>
 methods: {
    getsum (val) {
      alert(val);
    }
  }
  
// 子组件
<template>
  <div>
    <button @click="$emit('handler',123)">按钮</button>
  </div>
</template>
```
